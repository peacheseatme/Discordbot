1. Install Python .3.12.7. or greater.
2. Put your discord token in the .env file next to DISCORD_TOKEN=. This is used to grab the variuble exsternally and it is used as an ENV variuble instead.
3. Before proceding to the next step, please create a virtual enviroment using source .venv/bin/activate for installing required packages. (For crostini/linux. This will not work if one is not allready created.)
4. In a Python terminal, install contents of file requirements.txt.
5. Install bot to server by pasting URL into bar and authorizing the bot: https://discord.com/oauth2/authorize?client_id=1379645937064087563
6. Sign license.
7. Run "Bot.py"
